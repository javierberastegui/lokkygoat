# Identidad visual de Lokkygoat

## Dirección principal

Lokkygoat debe sentirse como una mascota protagonista de escritorio: amigable, tecnológica, viva y memorable.

La referencia conceptual es una mascota tipo robot adorable, espacial y expresiva, similar en espíritu a una mascota de videojuego moderna, pero con identidad propia de cabra-bot.

No debe copiar diseños existentes.
Debe inspirarse en la sensación de:

- robot pequeño y simpático;
- compañero de aventura;
- asistente flotante;
- mascota premium;
- bridge visual entre el usuario y varias IAs.

## Concepto

Lokkygoat no es una cabra literal.

Lokkygoat es una cabra-bot espacial:

- cuerpo blanco redondeado;
- visor negro brillante;
- ojos azules expresivos;
- cuernos pequeños de cabra;
- orejas o antenas laterales;
- detalles azul neón;
- animaciones suaves;
- energía de mascota viva;
- estética limpia, cute y tecnológica.

## Nombres internos

- Proyecto/app: `Lokkygoat`
- Mascota visible: `Goatky`
- Bridge interno: `Goatway`
- Cerebro IA: `Goatmind`
- Memoria/local state: `Goatmemory`

## Personalidad visual

Debe transmitir:

- “estoy vivo”;
- “soy tu compañero”;
- “puedo conectar con otras IAs”;
- “soy tecnológico, pero no intimidante”;
- “soy una mascota de escritorio, no solo una ventana de chat”.

## Estados visuales esperados

### Idle

Goatky flota suavemente, mira al usuario y parpadea.

### Thinking

El visor o núcleo se ilumina en azul/violeta.
Puede aparecer pulso suave.

### Success

Animación alegre, salto, giro o brillo.

### Error

Señal visual suave, sin parecer roto.
Color rojo o naranja controlado.

### Sleep

Ojos cerrados, flotación lenta, burbuja Zzz.

### Bridge active

Cuando actúa como puente hacia Hermes/OpenClaw/Ollama, debe sentirse como si estuviera “canalizando” energía o conectando mundos.

## Regla de implementación

La estética debe evolucionar sin romper la arquitectura.

Primero se documenta la identidad.
Luego se ajusta `pet.html` de forma progresiva.

No mezclar cambios visuales con providers, IPC o seguridad salvo que sea imprescindible.

## Siguiente paso visual recomendado

Crear una versión de Goatky más cercana a cabra-bot:

- visor negro con ojos azules;
- cuerpo blanco redondeado;
- cuernos minimalistas;
- orejas/antenas;
- aura azul/violeta;
- silueta más reconocible como mascota.
